# Netrum Lite Node Monitor

A professional, real-time dashboard for monitoring your Netrum Lite nodes. Built with React, Next.js, and Tailwind CSS, this tool provides a comprehensive overview of your node's performance and the overall health of the Netrum network.

## Key Features

- **Real-time Monitoring**: Track your node's CPU usage, RAM allocation, and network speeds (upload/download) in real-time.
- **Wallet Integration**: Seamlessly view your ETH and NPT balances directly on the dashboard.
- **Mining Statistics**: Monitor your mining status, current mining speed (NPT/hr), and total tokens mined.
- **Network Health**: stay informed with global network stats, including total active nodes, current block height, and gas prices.
- **Network Participation**: A detailed table showing all active participants across the Netrum network.
- **Customizable Experience**: Toggle between light and dark modes to suit your preference.
- **Responsive Design**: Optimized for both desktop and mobile viewing with a premium, glassmorphic UI.

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn
- A Netrum wallet address

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/Netrum-Node-Dashboard.git
   cd Netrum-Node-Dashboard
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Access the dashboard:**
   Open [http://localhost:3000](http://localhost:3000) in your browser.

## Configuration

To monitor your specific node, click the **Configure** button in the header and enter your:
- **Node ID**: Your unique lite node identifier.
- **Wallet Address**: Your public Netrum wallet address.

The dashboard will automatically remember your settings for future visits.
